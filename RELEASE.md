# Release Management

## Current Base Tag

- v1.0.0

## Latest Release Tag

- v1.1.0

## Process

1. Update this file with the new release tag.
2. Create an annotated git tag that matches the version.
3. Push the commit and tag to the remote.

## Tagging Commands

```bash
git tag -a v1.1.0 -m "Release v1.1.0"
git push origin v1.1.0
```
